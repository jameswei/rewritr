原生 macOS 写作助手

隐私优先， 开源透明的 英文改写工具。

Rewritr 帮助非英语母语用户 把选中的英文改写得更顺畅、 更自然、更接近母语表达， 就在你正在写作的地方完成。

不需要 Rewritr 托管账号。 不保存改写历史。不做分析统计。 使用你自己的 OpenAI-compatible provider， 也可以连接本地模型。

隐私优先

你的文本应该由你掌控。

Rewritr 刻意保持小而清晰。它不运行后端服务，不收集分析数据， 不保存改写历史，也不要求你创建账号。

无 Rewritr 后端

只发送选中的文本

应用只会把你明确选中的文本发送到你配置的 provider endpoint。

自带 provider

由你决定改写发生在哪里

可以使用 OpenAI-compatible 云端 endpoint，也可以使用本地模型， 让数据留在自己的机器或网络里。

macOS Keychain

API key 留在你的 Mac

Provider key 保存在 Keychain。非敏感设置保存在本地应用偏好设置中。

开源

可以检查它做了什么

Rewritr 免费且开源，信任模型是可见的，而不是藏在托管写作服务背后。

用途

更自然的日常英文，而不是新的写作平台。

Rewritr 面向已经知道自己想表达什么、只是希望最终英文更顺畅自然的人。

它不是文档编辑器、AI 内容生成器、学术写作套件、浏览器插件， 也不是订阅制写作账号。

01

选中文本

在你正在使用的应用里，选中想要改进的句子或段落。

02

触发 Rewritr

使用 macOS 菜单栏应用的全局快捷键。Rewritr 通过标准 macOS 自动化复制选区。

03

预览或替换

你可以先检查改写结果，也可以在需要最快流程时直接替换。

设计上的不同

更窄的工具，更少隐藏代价。

不局限在浏览器里

可用于支持标准复制粘贴的普通 macOS 应用和浏览器文本框。

不围绕账号运行

没有 Rewritr 登录、托管工作区、用量面板或服务端改写历史。

不是付费墙后的 AI 套件

应用只专注一件事：用你控制的 provider 自然地改写选中的英文。

不过度正式

默认改写目标是清晰、接近母语的日常英文，而不是学术或论文式表达。

兼容性

最适合普通可编辑文本。

Rewritr 最适合支持标准选区、⌘-C 和 ⌘-V 的应用和浏览器输入框。

安全输入框、Terminal、远程桌面、虚拟机、浏览器 IDE，以及焦点处理特殊的应用可能受限。
